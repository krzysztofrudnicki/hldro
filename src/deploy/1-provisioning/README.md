# Provisioning Infrastruktury

Katalog zawiera skrypty do provisioningu infrastruktury chmurowej.

## Cel

Tworzenie i konfiguracja podstawowych zasobów infrastrukturalnych:
- Grupy zasobów
- Konta storage
- Azure Functions
- Service Bus
- Bazy danych
- Inne zasoby Azure

## Narzędzia

- Terraform / Bicep / ARM templates
- Azure CLI
- Skrypty PowerShell/Bash

## Użycie

Skrypty w tym katalogu powinny być wykonywane jako pierwsze w procesie deploymentu.
