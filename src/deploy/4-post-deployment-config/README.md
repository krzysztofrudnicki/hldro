# Konfiguracja Post-Deployment

Katalog zawiera skrypty do konfiguracji środowiska po deploymencie aplikacji.

## Cel

Finalne kroki konfiguracyjne:
- Inicjalizacja bazy danych (seedowanie)
- Konfiguracja domen i certyfikatów
- Testy smoke
- Walidacja deploymentu
- Konfiguracja monitoringu aplikacyjnego
- Ustawienia production-ready

## Użycie

Skrypty wykonywane jako ostatnie w procesie deploymentu, po wdrożeniu aplikacji.
