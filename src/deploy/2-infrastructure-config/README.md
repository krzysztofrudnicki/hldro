# Konfiguracja Infrastruktury

Katalog zawiera skrypty do konfiguracji już utworzonej infrastruktury.

## Cel

Konfiguracja zasobów po ich utworzeniu:
- Ustawienia sieciowe (VNets, subnets, NSG)
- Konfiguracja zabezpieczeń
- Polityki dostępu
- Monitoring i alerting
- Konfiguracja połączeń między serwisami

## Użycie

Skrypty wykonywane po provisioning, przed deploymentem aplikacji.
