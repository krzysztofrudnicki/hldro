## Best Practices

1. **Aktualizuj dokumentację** przy każdej zmianie procesu
2. **Diagramy** - używaj Mermaid/PlantUML
3. **Przeglądy** - review dokumentacji co kwartał
4. **Dostępność** - trzymaj w repo, blisko kodu
